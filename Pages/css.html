CSS

CSS is a language that describes the style of an HTML document. br CSS describes how HTML elements should be displayed. It was created to separate the content of a document from its presentation. CSS is a language for describing the presentation of Web pages, including colors, layout, and fonts. Created by H√•kon Wium Lie, Bert Bos, and later the World Wide Web Consortium (W3C). It was created in 1994 and is now a W3C Recommendation. We have a variety of CSS properties that we can use to style our HTML elements. Like for example: color, background-color, font-size, font-family, text-align, etc. All this properties are used to style our HTML elements. The best part is that we can use CSS to style our HTML elements in any way we want. We can change the color of our text, the background color of our page, the font size, the font family, the text alignment, etc. We can even combine it with JavaScript to make our website more interactive. We have 3 ways of implementing CSS in our HTML document. We can use inline CSS, internal CSS, and external CSS. Inline CSS is when we use the style attribute in our HTML element to style it. Internal CSS is when we use the style tag in our HTML document to style our HTML elements. External CSS is when we use a separate CSS file to style our HTML elements.
